HTML & CSS Basic Summary

HTML Summary

HTML (HyperText Markup Language) is used to define our page contest, structure and meaning. You don't use it for styling purposes. Use CSS for that instead!

  • HTML uses "elements" to describe (annotate) contest
  • HTML elements typically have an opening tag, contest and the a closing
  • You can also have void (empty) elements with atributes
  • There's long list of available elements but you'll gain experinece over time, no worries

Learn more about all available HTML elements on the MDN HTML element reference.

CSS Summary

CSS (Cascading Styles Sheets) is used for styling your page contest.

  • Styles are assigned via property-value pairs
  • You can assign styles via the "styles" attribute
  • To avoid code duplication, you typically use global styles (e.g.via the "style" element) instead
  • a=Alternatively, you can work with external stylesheet files which you "link" to
  • When working wih CSS, concepts link "inheritance", "specificity" and the "box model" should be understood.

    • Learn more about all available CSS properties and values on the MDN CSS property reference.
